# **Angular 2**

Example of how to build a client for an API based on [ASP.NET Core](https://github.com/jealvarez/dotnet-example-api)

## **Prerequisites**

* NodeJs
* Bower
* TypeScript

# **Install Dependencies**

1. To install backend dependencies go to the main directory where is located the ```package.json``` 
```sh
$ npm install
```

2. To install frontend dependencies go to the main directory where is located the ```bower.json```
```sh
$ bower install
```

# **Execution**

On the main directory where is located the ```package.json```

2. To run
```sh
$ ng serve
```

3. Finally, [test it here!!](http://127.0.0.1:4200)
